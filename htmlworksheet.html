HTML Worksheet

HTML (Hypertext markup language) is the standar markup language for web pages.

Why is there need of HTML ?

HTML is standard markup language for Web Pages.

What are web pages ?

Web pages are pages comprising hypertexts created for websites over the network.

What is website ?

Website is a combination of two or more web pages containing visible information to users, a website contains a Homepage and several linked pages called web pages. Web Development is the industry for creating websites for internet or intranet.

What is Web Development

Web Development is the process of creating websites for internet (publicly) or intranet (privately) to serve web contetns to users.

What are the types of web Development ?

There are three types of web Development given below:

  1. Front-end Web Development
  2. Back-end Web Development
  3. Full-stack Web Development
Front-end Web Development - Front-end web development is the process of creating web pages for client-side interface, HTML,CSS & JS are used to create client-side user interface or web pages as front-end designing of the website.

Back-end Web Development - Back-end web development is the process of creating web pages for server-side runtime environment, PHP is the most used Back-end server-side scripting language over the web.

Full-stack Web Development - Full-stack web development is the combination of both the developments of client-side and server-side. A full-stack developer gains higher salary among all the web developers and designers.
